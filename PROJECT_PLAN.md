Project: Listify

What is Listify for?
Many task management applications become overly complex, requiring users to spend time learning features instead of focusing on completing their tasks. This can create unnecessary stress, clutter, and decision fatigue.

Listify aims to solve this problem by providing a simple, clean, and minimalist task management experience. The goal is to help users organize and manage their tasks without overwhelming them with excessive features or complicated interfaces.

The application will focus on essential task management functionality, allowing users to create, update, complete, and delete tasks in an intuitive way. By maintaining a clutter-free design and a straightforward workflow, Listify will help users stay organized, reduce mental overload, and focus on what matters most: getting things done.

Who uses it?
Students, professionals and anyone who wants a simple and easy-to-use design can manage and organise their tasks using Listify

Features:- 
Create tasks
Update tasks
View all tasks
Delete tasks
Mark tasks as completed

Possible future features:
Time blocking
Segregating tasks according to domains (academics, work, personal, etc)
prioritise tasks
add due dates for tasks
